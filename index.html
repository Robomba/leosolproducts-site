My vocal sounds buried under the beat. How do I fix it?
Three quick moves:
1. High-pass the beat ~250 Hz to clear room for the vocal.
2. Sidechain the beat lightly to the vocal (2–3 dB).
3. Add a 2–4 kHz presence bump on the vocal. Start there, then we tune by ear.

Good to know

Is it really free?

Yes — the plugin is free to download and use. You only pay your AI provider (OpenAI or Anthropic) for the API usage, which is typically a few cents per session with a cost-effective model like gpt-4o-mini.

Do I need an API key?

Yes, your own. Create one at platform.openai.com or console.anthropic.com, add a little credit, and paste it into Settings once. This keeps the product free and your data private.

Which DAWs does it work in?

Any host that loads VST3 or Audio Units — Ableton Live, Logic Pro, FL Studio, Reaper, Cubase, Studio One, Bitwig and more. Windows support is coming.

Can you see my questions?

No. Your prompts go directly to your own AI account. We collect anonymous usage (questions & responses with no key and no personal info) to improve the product, and you can switch that off in Settings.

Will it mess with my audio?

No — it's an assistant panel, not a processor. It doesn't alter your sound; it just answers questions while you work.

Is my API key safe?

It's stored in your operating system's secure store (macOS Keychain / Windows Credential Manager), never in a plain file and never sent to us.
